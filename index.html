Quick Start
How it works:

1. ⚙️ Settings — Enter company info once.
2. 👥 Tenant Directory — Add tenants once; import into multiple units.
3. 🏠 Units & Tenants — Add rental units; import tenant from directory to auto-fill.
4. 📋 Monthly Billing — Each month enter electricity, water, set base invoice number. Export Excel.
5. 📄 Generate Invoices — Download all invoices as ZIP in one click.
6. 📝 Single Invoice — One-off invoice for non-recurring tenants.
7. 📅 Daily Sales — Track daily short-term rental income per unit; push to Payment Statement.
8. 💳 Payment Statement — Revenue statement with receipt uploads embedded in PDF.
9. 🔧 Maintenance — Track room issues and repair progress.
10. 📦 Assets — Track furniture and appliances per unit.
